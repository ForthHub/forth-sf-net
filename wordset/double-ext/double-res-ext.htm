R.8 Double-Number reserved extension words
R.8.6.1.1830 M-
m-minus DOUBLE RES EXT 
        ( d1|ud1 n -- d2|ud2 )

subtract n from d1|ud1, giving the difference d2|ud2.

See: 6.1.1810 M*, 8.6.1.1830 M+, R.8.6.1.1810 M/,

R.8.6.1.1810 M/
m-divide DOUBLE RES EXT 
        ( d1 n2 -- n3 )

n3 is the signed quotient of d1 divided by n2.

See: 6.1.1810 M*, 8.6.1.1830 M+, R.8.6.1.1830 M-,
